# DiscordBot-360

## Installation

* Récupérer le projet

```bash
$ git clone https://github.com/TuxBoy/DiscordBot-360.git
```

* Installer les dépendances : 
```bash
* npm i
```

Renommer le config.js.dist en config.js et ajouter le token du bot

Démarrer le serveur node

```bash
$ node index.js --harmony
```
